# LinkedIn

| API | Method | Endpoint |
| --- | --- | --- |
| Profiles | `POST` | `/v1/linkedin/profiles` |
| Companies | `POST` | `/v1/linkedin/companies` |
| Posts | `POST` | `/v1/linkedin/posts` |
| Jobs | `POST` | `/v1/linkedin/jobs` |

Each endpoint accepts public LinkedIn URLs and uses the standard asynchronous task flow.
