# LinkedIn Profiles

```json
{
  "urls": ["https://www.linkedin.com/in/example"]
}
```

Results use `platform="linkedin"`, `resource="profiles"`, and `type="profile"`. Profile experience, education, location, and company details are returned under `extra`.
