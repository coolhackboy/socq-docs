# Facebook Posts

Collect public Facebook post data, including post text, source URLs, media fields, and engagement metrics.

## Endpoint

```http
POST https://api.socq.ai/v1/facebook/posts
```

## Request Body

```json
{
  "urls": ["https://www.facebook.com/facebook/posts/POST_ID"],
  "results_limit": 10
}
```

## Result Item
