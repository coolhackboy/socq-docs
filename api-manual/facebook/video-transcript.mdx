# Facebook Video Transcripts

```json
{"urls": ["https://www.facebook.com/reel/VIDEO_ID"]}
```

## Use with MCP

| Interface | Value |
| --- | --- |
| REST | `POST /v1/facebook/video-transcript` |
| Compact MCP | `socq_execute` with `endpoint: "facebook/video-transcript"` |
| Typed MCP | `socq_facebook_transcripts` |
| CLI | `socq facebook transcripts` |

Each available transcript is returned as normalized plain text.
