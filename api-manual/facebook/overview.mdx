# Facebook

Use the Facebook endpoints to collect public pages, posts, transcripts, group posts, comments, and events.

## Available Endpoints

| API | Method | Endpoint |
| --- | --- | --- |
| Facebook Pages | `POST` | `/v1/facebook/pages` |
| Facebook Posts | `POST` | `/v1/facebook/posts` |
| Facebook Comments | `POST` | `/v1/facebook/comments` |
| Facebook Video Transcripts | `POST` | `/v1/facebook/transcripts` |
| Facebook Group Posts | `POST` | `/v1/facebook/group-posts` |
| Facebook Events Search | `POST` | `/v1/facebook/events-search` |

## Workflow

Submit a request, receive a `task_id`, then poll `GET /v1/tasks/{task_id}`. When the task succeeds, the same response includes normalized results under `results`.
