# Facebook

Use the Facebook endpoints to collect public pages, posts, transcripts, group posts, comments, and events.

## Available Endpoints

| API | Method | Endpoint |
| --- | --- | --- |
| Facebook Pages | `POST` | `/v1/facebook/pages` |
| Facebook Posts | `POST` | `/v1/facebook/posts` |
| Facebook Comments | `POST` | `/v1/facebook/comments` |
| Facebook Video Transcripts | `POST` | `/v1/facebook/video-transcript` |
| Facebook Group Posts | `POST` | `/v1/facebook/group-posts` |
| Facebook Events Search | `POST` | `/v1/facebook/events-search` |
| Facebook Profiles | `POST` | `/v1/facebook/profiles` |
| Facebook Profile Photos | `POST` | `/v1/facebook/profile-photos` |
| Facebook Reels | `POST` | `/v1/facebook/reels` |
| Facebook Comment Replies | `POST` | `/v1/facebook/comment-replies` |
| Facebook Profile Events | `POST` | `/v1/facebook/profile-events` |
| Facebook Event Details | `POST` | `/v1/facebook/event-details` |
| Facebook Ad Transcript | `POST` | `/v1/facebook/ad-transcript` |
| Facebook Company Reviews | `POST` | `/v1/facebook/company-reviews` |

## Workflow

Submit a request, receive a `task_id`, then poll `GET /v1/tasks/{task_id}`. When the task succeeds, the same response includes normalized results under `results`.
