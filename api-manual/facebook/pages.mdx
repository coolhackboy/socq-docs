# Facebook Pages

Collect public Facebook page/profile metadata, public URLs, profile fields, and related public metrics.

## Endpoint

```http
POST https://api.socq.ai/v1/facebook/pages
```

## Request Body

```json
{
  "urls": ["https://www.facebook.com/facebook"]
}
```

{/* mcp-usage:start */}
## Use with MCP

| Interface | Value |
| --- | --- |
| REST | `POST /v1/facebook/pages` |
| Compact MCP | `socq_execute` with `endpoint: "facebook/pages"` |
| Typed MCP | `socq_facebook_pages` |
| Exact-tool URL | `https://api.socq.ai/mcp?tools=facebook_pages` |
| CLI | `socq facebook pages` |

Use the JSON request body above as the nested `input` object for `socq_execute`, or pass the same fields directly to the typed tool. Typed tools also accept `_wait_seconds`, `_result_limit`, and `_idempotency_key`.

<Info>
  A successful collection returns a `task_id`. Continue with `socq_get_task` until the task succeeds, then read normalized records from `results.items`.
</Info>

See [MCP quickstart](/integrations/mcp) for the complete submission and polling flow.
{/* mcp-usage:end */}

## Result Item

After the task succeeds, call `GET /v1/tasks/{task_id}` to read paginated standard records under `results`.

```json
{
  "id": "100064860875397",
  "platform": "facebook",
  "resource": "pages",
  "type": "page",
  "name": "Facebook",
  "url": "https://www.facebook.com/facebook/",
  "category": "Internet company",
  "about": "Public page description",
  "avatar_url": "https://example.com/avatar.jpg",
  "cover_url": "https://example.com/cover.jpg",
  "metrics": {
    "followers_count": 154000000,
    "likes_count": 154831857,
    "reviews_count": null,
    "rating": null
  },
  "contact": {
    "website": "https://example.com",
    "email": null,
    "phone": null,
    "address": null
  },
  "created_at": "2007-11-07T00:00:00",
  "collected_at": "2026-07-08T08:59:45",
  "extra": {}

```

You can pass multiple page URLs in the same field:

```json
{
  "urls": [
    "https://www.facebook.com/facebook",
    "https://www.facebook.com/Meta"
  ]
}
```
