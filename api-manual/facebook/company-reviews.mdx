# Facebook Company Reviews

Collect public Facebook company reviews. Results are normalized and billed by the actual saved record count.

```json
{
  "urls": [
    "https://www.facebook.com/PepsiCo/reviews/"
  ],
  "results_limit": 20
}
```

{/* mcp-usage:start */}
## Use with MCP

| Interface | Value |
| --- | --- |
| REST | `POST /v1/facebook/company-reviews` |
| Compact MCP | `socq_execute` with `endpoint: "facebook/company-reviews"` |
| Typed MCP | `socq_facebook_company_reviews` |
| Exact-tool URL | `https://api.socq.ai/mcp?tools=facebook_company_reviews` |
| CLI | `socq facebook company-reviews` |

Use the JSON request body above as the nested `input` object for `socq_execute`, or pass the same fields directly to the typed tool. Typed tools also accept `_wait_seconds`, `_result_limit`, and `_idempotency_key`.

<Info>
  A successful submission returns a `task_id`. Continue with `socq_get_task` until the task succeeds, then read normalized records from `results.items`.
</Info>

See [MCP quickstart](/integrations/mcp) for the complete submission and polling flow.
{/* mcp-usage:end */}
