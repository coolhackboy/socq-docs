# List Task Results

Returns cursor-paginated standard records for a task. Each product API has its own stable result shape, but all records use business-level field names instead of provider-specific field names.

## Endpoint

```http
GET https://api.socq.ai/v1/tasks/{task_id}/results?limit=50
```

Use the `next_cursor` returned by one response to read the next page:

```http
GET https://api.socq.ai/v1/tasks/{task_id}/results?limit=50&cursor=NTY3ODkw
```

## Pagination

| Query | Type | Default | Description |
| --- | --- | --- | --- |
| `limit` | integer | `50` | Number of records to return, max `500` |
| `cursor` | string | `null` | Cursor from the previous response. Omit it for the first page. |
