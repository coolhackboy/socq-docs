# Company Search

Search for public companies and retrieve the page ID required by Company Ads.

## Request Example

```json
{
  "query": "Nike"
}
```

{/* mcp-usage:start */}
## Use with MCP

| Interface | Value |
| --- | --- |
| REST | `POST /v1/facebook-ad-library/company-search` |
| Compact MCP | `socq_execute` with `endpoint: "facebook-ad-library/company-search"` |
| Typed MCP | `socq_facebook_ad_library_company_search` |
| Exact-tool URL | `https://api.socq.ai/mcp?tools=facebook_ad_library_company_search` |
| CLI | `socq facebook-ad-library company-search` |

Use the JSON request body above as the nested `input` object for `socq_execute`, or pass the same fields directly to the typed tool. Typed tools also accept `_wait_seconds`, `_result_limit`, and `_idempotency_key`.

<Info>
  A successful collection returns a `task_id`. Continue with `socq_get_task` until the task succeeds, then read normalized records from `results.items`.
</Info>

See [MCP quickstart](/integrations/mcp) for the complete submission and polling flow.
{/* mcp-usage:end */}

This endpoint saves every company candidate returned by the search request.
