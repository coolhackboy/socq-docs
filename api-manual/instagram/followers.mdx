# Instagram Followers

```json
{"usernames": ["instagram"], "results_limit": 100}
```

## Use with MCP

| Interface | Value |
| --- | --- |
| REST | `POST /v1/instagram/followers` |
| Compact MCP | `socq_execute` with `endpoint: "instagram/followers"` |
| Typed MCP | `socq_instagram_followers` |
| CLI | `socq instagram followers` |

Only relationships visible for public profiles can be returned.
