# Instagram Tagged Posts

```json
{"usernames": ["instagram"], "results_limit": 20}
```

## Use with MCP

| Interface | Value |
| --- | --- |
| REST | `POST /v1/instagram/tagged-posts` |
| Compact MCP | `socq_execute` with `endpoint: "instagram/tagged-posts"` |
| Typed MCP | `socq_instagram_tagged_posts` |
| CLI | `socq instagram tagged-posts` |

Results use the normalized Instagram post shape and include available tags, mentions, media, and engagement metrics.
