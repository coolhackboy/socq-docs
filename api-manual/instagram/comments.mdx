# Instagram Comments

Collect public Instagram comment data for public post URLs, including comment text, author fields, engagement metrics, and raw JSONL output.

## Endpoint

```http
POST https://api.socq.ai/v1/instagram/comments
```

## Request Body

```json
{
  "urls": ["https://www.instagram.com/p/POST_CODE/"]
}
```

{/* mcp-usage:start */}
## Use with MCP

| Interface | Value |
| --- | --- |
| REST | `POST /v1/instagram/comments` |
| Compact MCP | `socq_execute` with `endpoint: "instagram/comments"` |
| Typed MCP | `socq_instagram_comments` |
| Exact-tool URL | `https://api.socq.ai/mcp?tools=instagram_comments` |
| CLI | `socq instagram comments` |

Use the JSON request body above as the nested `input` object for `socq_execute`, or pass the same fields directly to the typed tool. Typed tools also accept `_wait_seconds`, `_result_limit`, and `_idempotency_key`.

<Info>
  A successful collection returns a `task_id`. Continue with `socq_get_task` until the task succeeds, then read normalized records from `results.items`.
</Info>

See [MCP quickstart](/integrations/mcp) for the complete submission and polling flow.
{/* mcp-usage:end */}

All returned comments are billed by the actual result count. This endpoint does not accept a result-count limit.
