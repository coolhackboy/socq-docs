# Instagram Posts

Collect public Instagram post data, including caption text, engagement metrics, media URLs, author fields, and raw JSONL output.

## Endpoint

```http
POST https://api.socq.ai/v1/instagram/posts
```

## Request Body

```json
{
  "usernames": ["instagram"],
  "results_limit": 10
}
```

{/* mcp-usage:start */}
## Use with MCP

| Interface | Value |
| --- | --- |
| REST | `POST /v1/instagram/posts` |
| Compact MCP | `socq_execute` with `endpoint: "instagram/posts"` |
| Typed MCP | `socq_instagram_posts` |
| Exact-tool URL | `https://api.socq.ai/mcp?tools=instagram_posts` |
| CLI | `socq instagram posts` |

Use the JSON request body above as the nested `input` object for `socq_execute`, or pass the same fields directly to the typed tool. Typed tools also accept `_wait_seconds`, `_result_limit`, and `_idempotency_key`.

<Info>
  A successful collection returns a `task_id`. Continue with `socq_get_task` until the task succeeds, then read normalized records from `results.items`.
</Info>

See [MCP quickstart](/integrations/mcp) for the complete submission and polling flow.
{/* mcp-usage:end */}
