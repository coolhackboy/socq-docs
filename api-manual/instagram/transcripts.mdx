# Instagram Transcripts

```json
{"urls": ["https://www.instagram.com/reel/SHORTCODE/"]}
```

## Use with MCP

| Interface | Value |
| --- | --- |
| REST | `POST /v1/instagram/transcripts` |
| Compact MCP | `socq_execute` with `endpoint: "instagram/transcripts"` |
| Typed MCP | `socq_instagram_transcripts` |
| CLI | `socq instagram transcripts` |

Carousel posts may produce more than one transcript record. Media without speech may return no result.
