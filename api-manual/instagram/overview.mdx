# Instagram

Use the Instagram endpoints to collect public profiles, relationship lists, posts, transcripts, and discovery data.

## Available Endpoints

| API | Method | Endpoint |
| --- | --- | --- |
| Instagram Posts | `POST` | `/v1/instagram/posts` |
| Instagram Comments | `POST` | `/v1/instagram/comments` |
| Instagram Reels | `POST` | `/v1/instagram/reels` |
| Instagram Search | `POST` | `/v1/instagram/search` |
| Instagram Followers Count | `POST` | `/v1/instagram/followers-count` |
| Instagram Profiles | `POST` | `/v1/instagram/profiles` |
| Instagram Transcripts | `POST` | `/v1/instagram/transcript` |
| Instagram Hashtag Posts | `POST` | `/v1/instagram/hashtag-posts` |
| Instagram Followers | `POST` | `/v1/instagram/followers-list` |
| Instagram Following | `POST` | `/v1/instagram/following-list` |
| Instagram Tagged Posts | `POST` | `/v1/instagram/tagged-posts` |
| Instagram Reels Search | `POST` | `/v1/instagram/reels-search` |
| Instagram Audio Reels | `POST` | `/v1/instagram/audio-reels` |
| Instagram Post Info | `POST` | `/v1/instagram/post-info` |
| Instagram Trending Reels | `POST` | `/v1/instagram/trending-reels` |
| Instagram Story Highlights | `POST` | `/v1/instagram/story-highlights` |
| Instagram Highlight Items | `POST` | `/v1/instagram/highlight-items` |

## Workflow

<Steps>
  <Step title="Submit">
    Send a request to an Instagram endpoint.
  </Step>
  <Step title="Receive task ID">
    The API returns a `task_id` immediately.
  </Step>
  <Step title="Check status">
    Poll `GET /v1/tasks/{task_id}` until the task reaches `succeeded` or `failed`.
  </Step>
  <Step title="Read data">
    When the task succeeds, `/v1/tasks/{task_id}` returns normalized rows under `results`. Download raw JSONL from `files` when needed.
  </Step>
</Steps>
