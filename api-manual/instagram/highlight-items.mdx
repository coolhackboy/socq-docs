# Instagram Highlight Items

Collect items from public Instagram highlights. Results are normalized and billed by the actual saved record count.

```json
{
  "highlight_ids": [
    "18067016518767507"
  ]
}
```

{/* mcp-usage:start */}
## Use with MCP

| Interface | Value |
| --- | --- |
| REST | `POST /v1/instagram/highlight-items` |
| Compact MCP | `socq_execute` with `endpoint: "instagram/highlight-items"` |
| Typed MCP | `socq_instagram_highlight_items` |
| Exact-tool URL | `https://api.socq.ai/mcp?tools=instagram_highlight_items` |
| CLI | `socq instagram highlight-items` |

Use the JSON request body above as the nested `input` object for `socq_execute`, or pass the same fields directly to the typed tool. Typed tools also accept `_wait_seconds`, `_result_limit`, and `_idempotency_key`.

<Info>
  A successful submission returns a `task_id`. Continue with `socq_get_task` until the task succeeds, then read normalized records from `results.items`.
</Info>

See [MCP quickstart](/integrations/mcp) for the complete submission and polling flow.
{/* mcp-usage:end */}
