# YouTube Channel Videos

Collect regular videos from one or more public channels.

```json
{
  "urls": ["https://www.youtube.com/@example"],
  "results_limit": 100
}
```

`results_limit` defaults to 100 and applies to each channel URL.

Results use the normalized video structure documented in [YouTube Videos](/api-manual/youtube/videos), with `resource` set to `channel-videos`.
