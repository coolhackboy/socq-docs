# YouTube Comments

Collect comments from one or more public video URLs.

```json
{
  "urls": ["https://www.youtube.com/watch?v=VIDEO_ID"],
  "results_limit": 100,
  "sort_by": "newest"
}
```

{/* mcp-usage:start */}
## Use with MCP

| Interface | Value |
| --- | --- |
| REST | `POST /v1/youtube/comments` |
| Compact MCP | `socq_execute` with `endpoint: "youtube/comments"` |
| Typed MCP | `socq_youtube_comments` |
| Exact-tool URL | `https://api.socq.ai/mcp?tools=youtube_comments` |
| CLI | `socq youtube comments` |

Use the JSON request body above as the nested `input` object for `socq_execute`, or pass the same fields directly to the typed tool. Typed tools also accept `_wait_seconds`, `_result_limit`, and `_idempotency_key`.

<Info>
  A successful collection returns a `task_id`. Continue with `socq_get_task` until the task succeeds, then read normalized records from `results.items`.
</Info>

See [MCP quickstart](/integrations/mcp) for the complete submission and polling flow.
{/* mcp-usage:end */}

`results_limit` defaults to 100 per video. `sort_by` accepts `newest` or `relevance`.
