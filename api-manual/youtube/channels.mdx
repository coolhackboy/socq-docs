# YouTube Channels

Collect channel metadata for one or more public channel URLs.

```json
{
  "urls": ["https://www.youtube.com/@example"]
}
```

## Result Item

```json
{
  "id": "UC123456789",
  "platform": "youtube",
  "resource": "channels",
  "type": "channel",
  "username": "example",
  "name": "Example Channel",
  "url": "https://www.youtube.com/@example",
  "description": "Public channel description",
  "avatar_url": "https://example.com/avatar.jpg",
  "banner_url": "https://example.com/banner.jpg",
  "is_verified": true,
  "metrics": {
    "subscribers_count": 125000,
    "videos_count": 240,
    "views_count": 18000000
  },
  "created_at": "2017-01-04T00:00:00",
  "collected_at": "2026-07-15T10:30:00",
  "extra": {}
}
```
