# Pinterest User Pins

```json
{
  "urls": ["https://www.pinterest.com/example/"],
  "results_limit": 100
}
```

`results_limit` defaults to 100, accepts values from 1 through 2,000, and applies to each profile URL. Results are Pins, not profile records.
