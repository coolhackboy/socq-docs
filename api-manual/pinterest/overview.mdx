# Pinterest

| API | Method | Endpoint |
| --- | --- | --- |
| Profiles | `POST` | `/v1/pinterest/profiles` |
| Pins | `POST` | `/v1/pinterest/pins` |
| User Pins | `POST` | `/v1/pinterest/user-pins` |
| Search | `POST` | `/v1/pinterest/search` |

Profiles return account metadata. Pins, User Pins, and Search return normalized Pin records.
