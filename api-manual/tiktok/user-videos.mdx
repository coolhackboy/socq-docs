# TikTok User Videos

Collect public videos posted by TikTok users. Results are normalized and billed by the actual saved record count.

```json
{
  "usernames": [
    "@tiktok"
  ],
  "results_limit": 20
}
```

{/* mcp-usage:start */}
## Use with MCP

| Interface | Value |
| --- | --- |
| REST | `POST /v1/tiktok/user-videos` |
| Compact MCP | `socq_execute` with `endpoint: "tiktok/user-videos"` |
| Typed MCP | `socq_tiktok_user_videos` |
| Exact-tool URL | `https://api.socq.ai/mcp?tools=tiktok_user_videos` |
| CLI | `socq tiktok user-videos` |

Use the JSON request body above as the nested `input` object for `socq_execute`, or pass the same fields directly to the typed tool. Typed tools also accept `_wait_seconds`, `_result_limit`, and `_idempotency_key`.

<Info>
  A successful submission returns a `task_id`. Continue with `socq_get_task` until the task succeeds, then read normalized records from `results.items`.
</Info>

See [MCP quickstart](/integrations/mcp) for the complete submission and polling flow.
{/* mcp-usage:end */}
