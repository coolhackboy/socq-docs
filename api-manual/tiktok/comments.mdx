# TikTok Comments

Collect top-level comments from one or more public video URLs.

```json
{
  "urls": ["https://www.tiktok.com/@example/video/7234567890123456789"],
  "results_limit": 100
}
```

`results_limit` defaults to 100 and applies to each video URL.
