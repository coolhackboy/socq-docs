# TikTok Comments

Collect top-level comments from one or more public video URLs.

```json
{
  "urls": ["https://www.tiktok.com/@example/video/7234567890123456789"],
  "results_limit": 100
}
```

{/* mcp-usage:start */}
## Use with MCP

| Interface | Value |
| --- | --- |
| REST | `POST /v1/tiktok/comments` |
| Compact MCP | `socq_execute` with `endpoint: "tiktok/comments"` |
| Typed MCP | `socq_tiktok_comments` |
| Exact-tool URL | `https://api.socq.ai/mcp?tools=tiktok_comments` |
| CLI | `socq tiktok comments` |

Use the JSON request body above as the nested `input` object for `socq_execute`, or pass the same fields directly to the typed tool. Typed tools also accept `_wait_seconds`, `_result_limit`, and `_idempotency_key`.

<Info>
  A successful collection returns a `task_id`. Continue with `socq_get_task` until the task succeeds, then read normalized records from `results.items`.
</Info>

See [MCP quickstart](/integrations/mcp) for the complete submission and polling flow.
{/* mcp-usage:end */}

`results_limit` defaults to 100 and applies to each video URL.
