# TikTok Following List

Collect public TikTok following profiles. Results are normalized and billed by the actual saved record count.

```json
{
  "usernames": [
    "@tiktok"
  ],
  "results_limit": 20
}
```

{/* mcp-usage:start */}
## Use with MCP

| Interface | Value |
| --- | --- |
| REST | `POST /v1/tiktok/following-list` |
| Compact MCP | `socq_execute` with `endpoint: "tiktok/following-list"` |
| Typed MCP | `socq_tiktok_following_list` |
| Exact-tool URL | `https://api.socq.ai/mcp?tools=tiktok_following_list` |
| CLI | `socq tiktok following-list` |

Use the JSON request body above as the nested `input` object for `socq_execute`, or pass the same fields directly to the typed tool. Typed tools also accept `_wait_seconds`, `_result_limit`, and `_idempotency_key`.

<Info>
  A successful submission returns a `task_id`. Continue with `socq_get_task` until the task succeeds, then read normalized records from `results.items`.
</Info>

See [MCP quickstart](/integrations/mcp) for the complete submission and polling flow.
{/* mcp-usage:end */}
