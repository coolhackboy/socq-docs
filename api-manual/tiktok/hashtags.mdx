# TikTok Hashtags

Collect public videos for one or more hashtags. A leading `#` is optional.

```json
{
  "hashtags": ["tutorial", "#python"],
  "results_limit": 50
}
```

{/* mcp-usage:start */}
## Use with MCP

| Interface | Value |
| --- | --- |
| REST | `POST /v1/tiktok/hashtags` |
| Compact MCP | `socq_execute` with `endpoint: "tiktok/hashtags"` |
| Typed MCP | `socq_tiktok_hashtags` |
| Exact-tool URL | `https://api.socq.ai/mcp?tools=tiktok_hashtags` |
| CLI | `socq tiktok hashtags` |

Use the JSON request body above as the nested `input` object for `socq_execute`, or pass the same fields directly to the typed tool. Typed tools also accept `_wait_seconds`, `_result_limit`, and `_idempotency_key`.

<Info>
  A successful collection returns a `task_id`. Continue with `socq_get_task` until the task succeeds, then read normalized records from `results.items`.
</Info>

See [MCP quickstart](/integrations/mcp) for the complete submission and polling flow.
{/* mcp-usage:end */}

`results_limit` defaults to 100 and applies to each hashtag.

Results use the normalized structure documented in [TikTok Videos](/api-manual/tiktok/videos), with `resource` set to `hashtags`. The matched hashtag and its available metrics are returned under `extra`.
