# TikTok Trending Feed

```json
{"region": "US", "results_limit": 20}
```

## Use with MCP

| Interface | Value |
| --- | --- |
| REST | `POST /v1/tiktok/trending-feed` |
| Compact MCP | `socq_execute` with `endpoint: "tiktok/trending-feed"` |
| Typed MCP | `socq_tiktok_trending_feed` |
| CLI | `socq tiktok trending-feed` |

`region` is a two-letter region code used to localize feed availability.
`results_limit` defaults to 100 and accepts values from 4 through 2,000.
