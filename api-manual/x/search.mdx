# X Search

Search public posts using a keyword or search expression.

```json
{
  "query": "AI lang:en",
  "results_limit": 100,
  "sort_by": "latest"
}
```

| Field | Accepted values |
| --- | --- |
| `sort_by` | `latest`, `top` |

Results use the normalized structure documented in [X Posts](/api-manual/x/posts), with `resource` set to `search`. The submitted expression is returned under `extra.search_query`.
