# X 搜索

使用关键字或搜索表达式搜索公共帖子。

```json
{
  "query": "AI lang:en",
  "results_limit": 100,
  "sort_by": "latest"
}
```

{/* mcp-usage:start */}
## 与 MCP 一起使用

| 接口 | 值 |
| --- | --- |
| REST | `POST /v1/x/search` |
| Compact MCP | `socq_execute` 与 `endpoint: "x/search"` |
| Typed MCP | `socq_x_search` |
| 精确工具 URL | `https://api.socq.ai/mcp?tools=x_search` |
| CLI | `socq x search` |

使用上面的 JSON 请求正文作为 `socq_execute` 的嵌套 `input` 对象，或将相同的字段直接传递给Typed 工具。Typed 工具还接受 `_wait_seconds`、`_result_limit` 和 `_idempotency_key`。

<Info>
  成功的采集将返回 `task_id`。继续 `socq_get_task` 直到任务成功，然后从 `results.items` 读取规范化记录。
</Info>

有关完整的提交和轮询流程，请参阅[MCP 快速入门](/zh/integrations/mcp)。
{/* mcp-usage:end */}

|领域|接受的值|
| --- | --- |
| `sort_by` | `latest`、`top` |

结果使用 [X 帖子](/zh/api-manual/x/posts) 中记录的归一化结构，其中 `resource` 设置为 `search`。提交的表达式在 `extra.search_query` 下返回。
