# X API

X 端点使用与其他代理 API 相同的异步任务流。

|API | 方法 | 端点 |
| --- | --- | --- |
|个人资料 | `POST` | `/v1/x/profiles` |
|帖子 | `POST` | `/v1/x/posts` |
|用户帖子 | `POST` | `/v1/x/user-posts` |
|搜索 | `POST` | `/v1/x/search` |

提交请求，收到 `task_id`，然后轮询 `GET /v1/tasks/{task_id}`。成功的任务包括 `results` 下的规范化记录。
