# X Followers List

采集公开的 X 粉丝资料。结果会转换为统一结构，并按实际保存的记录数计费。

```json
{
  "usernames": [
    "@X"
  ],
  "results_limit": 200
}
```

{/* mcp-usage:start */}
## 通过 MCP 使用

| Interface | Value |
| --- | --- |
| REST | `POST /v1/x/followers-list` |
| Compact MCP | `socq_execute` with `endpoint: "x/followers-list"` |
| Typed MCP | `socq_x_followers_list` |
| Exact-tool URL | `https://api.socq.ai/mcp?tools=x_followers_list` |
| CLI | `socq x followers-list` |

使用 `socq_execute` 时，将上面的 JSON 请求体作为 `input` 对象传入；也可以将相同字段直接传给 Typed MCP 工具。Typed MCP 工具还支持 `_wait_seconds`、`_result_limit` 和 `_idempotency_key`。

<Info>
  提交成功后会返回 `task_id`。继续调用 `socq_get_task` 直到任务成功，然后从 `results.items` 读取标准化记录。
</Info>

完整的提交和轮询流程请参阅 [MCP 快速开始](/zh/integrations/mcp)。
{/* mcp-usage:end */}
