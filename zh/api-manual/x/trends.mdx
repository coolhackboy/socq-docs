# X Trends

按 WOEID 采集公开的 X 趋势。结果会转换为统一结构，并按实际保存的记录数计费。

```json
{
  "woeids": [
    "1"
  ],
  "results_limit": 100
}
```

{/* mcp-usage:start */}
## 通过 MCP 使用

| Interface | Value |
| --- | --- |
| REST | `POST /v1/x/trends` |
| Compact MCP | `socq_execute` with `endpoint: "x/trends"` |
| Typed MCP | `socq_x_trends` |
| Exact-tool URL | `https://api.socq.ai/mcp?tools=x_trends` |
| CLI | `socq x trends` |

使用 `socq_execute` 时，将上面的 JSON 请求体作为 `input` 对象传入；也可以将相同字段直接传给 Typed MCP 工具。Typed MCP 工具还支持 `_wait_seconds`、`_result_limit` 和 `_idempotency_key`。

<Info>
  提交成功后会返回 `task_id`。继续调用 `socq_get_task` 直到任务成功，然后从 `results.items` 读取标准化记录。
</Info>

完整的提交和轮询流程请参阅 [MCP 快速开始](/zh/integrations/mcp)。
{/* mcp-usage:end */}
