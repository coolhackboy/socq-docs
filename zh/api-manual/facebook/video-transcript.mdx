# Facebook 视频字幕

```json
{"urls": ["https://www.facebook.com/reel/VIDEO_ID"]}
```

## 与 MCP 一起使用

| 接口 | 值 |
| --- | --- |
| REST | `POST /v1/facebook/video-transcript` |
| Compact MCP | `socq_execute` 与 `endpoint: "facebook/video-transcript"` |
| Typed MCP | `socq_facebook_transcripts` |
| CLI | `socq facebook transcripts` |

每个可用的转录本都以标准化纯文本形式返回。
