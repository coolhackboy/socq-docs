# Facebook

使用 Facebook 端点采集公共页面、帖子、视频字幕、群组帖子、评论和事件。

## 可用端点

|API | 方法 | 端点 |
| --- | --- | --- |
| Facebook 公共主页 | `POST` | `/v1/facebook/pages` |
| Facebook 帖子 | `POST` | `/v1/facebook/posts` |
| Facebook 评论 | `POST` | `/v1/facebook/comments` |
| Facebook 视频脚本 | `POST` | `/v1/facebook/video-transcript` |
| Facebook 群组帖子 | `POST` | `/v1/facebook/group-posts` |
| Facebook 活动搜索 | `POST` | `/v1/facebook/events-search` |
| Facebook 个人资料 | `POST` | `/v1/facebook/profiles` |
| Facebook 个人资料照片 | `POST` | `/v1/facebook/profile-photos` |
| Facebook Reels | `POST` | `/v1/facebook/reels` |
| Facebook 评论回复 | `POST` | `/v1/facebook/comment-replies` |
| Facebook 个人资料活动 | `POST` | `/v1/facebook/profile-events` |
| Facebook 活动详情 | `POST` | `/v1/facebook/event-details` |
| Facebook 广告转录 | `POST` | `/v1/facebook/ad-transcript` |
| Facebook 公司评论 | `POST` | `/v1/facebook/company-reviews` |

## 工作流程

提交请求，收到 `task_id`，然后轮询 `GET /v1/tasks/{task_id}`。当任务成功时，相同的响应包括 `results` 下的标准化结果。
