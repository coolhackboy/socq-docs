# Facebook 帖子

采集公共 Facebook 帖子数据，包括帖子文本、源 URL、媒体字段和参与度指标。

## 端点

```http
POST https://api.socq.ai/v1/facebook/posts
```

## 请求正文

```json
{
  "urls": ["https://www.facebook.com/facebook/posts/POST_ID"],
  "results_limit": 10
}
```

{/* mcp-usage:start */}
## 与 MCP 一起使用

| 接口 | 值 |
| --- | --- |
| REST | `POST /v1/facebook/posts` |
| Compact MCP | `socq_execute` 与 `endpoint: "facebook/posts"` |
| Typed MCP | `socq_facebook_posts` |
| 精确工具 URL | `https://api.socq.ai/mcp?tools=facebook_posts` |
| CLI | `socq facebook posts` |

使用上面的 JSON 请求正文作为 `socq_execute` 的嵌套 `input` 对象，或将相同的字段直接传递给Typed 工具。Typed 工具还接受 `_wait_seconds`、`_result_limit` 和 `_idempotency_key`。

<Info>
  成功的采集将返回 `task_id`。继续 `socq_get_task` 直到任务成功，然后从 `results.items` 读取规范化记录。
</Info>

有关完整的提交和轮询流程，请参阅[MCP 快速入门](/zh/integrations/mcp)。
{/* mcp-usage:end */}
