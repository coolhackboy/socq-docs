# YouTube 播放列表视频

从一个或多个公共播放列表采集标准化视频记录。

```json
{"urls": ["https://www.youtube.com/playlist?list=PLAYLIST_ID"], "results_limit": 100}
```

## 与 MCP 一起使用

| 接口 | 值 |
| --- | --- |
| REST | `POST /v1/youtube/playlist-videos` |
| Compact MCP | `socq_execute` 与 `endpoint: "youtube/playlist-videos"` |
| Typed MCP | `socq_youtube_playlist_videos` |
| CLI | `socq youtube playlist-videos` |

播放列表ID可以直接在`urls`中传递。结果包括 `extra` 下的播放列表上下文。
`results_limit` 默认为 100，接受 10 到 2,000 之间的值，并应用于每个播放列表。
