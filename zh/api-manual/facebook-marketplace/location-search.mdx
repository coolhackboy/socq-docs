# Marketplace 位置搜索

使用城市、地区或地名来查找Marketplace 搜索的位置坐标。

该端点不接受`results_limit`。任务保存为查询返回的每个候选位置。

## 请求示例

```json
{
  "query": "Austin"
}
```

{/* mcp-usage:start */}
## 与 MCP 一起使用

| 接口 | 值 |
| --- | --- |
| REST | `POST /v1/facebook-marketplace/location-search` |
| Compact MCP | `socq_execute` 与 `endpoint: "facebook-marketplace/location-search"` |
| Typed MCP | `socq_facebook_marketplace_location_search` |
| 精确工具 URL | `https://api.socq.ai/mcp?tools=facebook_marketplace_location_search` |
| CLI | `socq facebook-marketplace location-search` |

使用上面的 JSON 请求正文作为 `socq_execute` 的嵌套 `input` 对象，或将相同的字段直接传递给Typed 工具。Typed 工具还接受 `_wait_seconds`、`_result_limit` 和 `_idempotency_key`。

<Info>
  成功的采集将返回 `task_id`。继续 `socq_get_task` 直到任务成功，然后从 `results.items` 读取规范化记录。
</Info>

有关完整的提交和轮询流程，请参阅[MCP 快速入门](/zh/integrations/mcp)。
{/* mcp-usage:end */}
