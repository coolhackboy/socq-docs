# Instagram 帖子详情

采集公开 Instagram 帖子的详细数据。结果会转换为统一结构，并按实际保存的记录数计费。

```json
{
  "urls": [
    "https://www.instagram.com/p/ABC123xyz/"
  ]
}
```

{/* mcp-usage:start */}
## 通过 MCP 使用

| 接口 | 值 |
| --- | --- |
| REST | `POST /v1/instagram/post-info` |
| Compact MCP | `socq_execute`，并设置 `endpoint: "instagram/post-info"` |
| 类型化 MCP | `socq_instagram_post_info` |
| 精确工具 URL | `https://api.socq.ai/mcp?tools=instagram_post_info` |
| CLI | `socq instagram post-info` |

使用 `socq_execute` 时，将上面的 JSON 请求体作为 `input` 对象传入；也可以将相同字段直接传给类型化 MCP 工具。类型化 MCP 工具还支持 `_wait_seconds`、`_result_limit` 和 `_idempotency_key`。

<Info>
  提交成功后会返回 `task_id`。继续调用 `socq_get_task` 直到任务成功，然后从 `results.items` 读取标准化记录。
</Info>

完整的提交和轮询流程请参阅 [MCP 快速开始](/zh/integrations/mcp)。
{/* mcp-usage:end */}
