# Instagram 关注者

```json
{"usernames": ["instagram"], "results_limit": 100}
```

## 与 MCP 一起使用

| 接口 | 值 |
| --- | --- |
| REST | `POST /v1/instagram/followers-list` |
| Compact MCP | `socq_execute` 与 `endpoint: "instagram/followers-list"` |
| Typed MCP | `socq_instagram_followers` |
| CLI | `socq instagram followers` |

只能返回对公共个人资料可见的关系。
