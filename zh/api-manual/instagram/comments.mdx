# Instagram 评论

采集公共帖子 URL 的公共 Instagram 评论数据，包括评论文本、作者字段、参与度指标和原始 JSONL 输出。

## 端点

```http
POST https://api.socq.ai/v1/instagram/comments
```

## 请求正文

```json
{
  "urls": ["https://www.instagram.com/p/POST_CODE/"]
}
```

{/* mcp-usage:start */}
## 与 MCP 一起使用

| 接口 | 值 |
| --- | --- |
| REST | `POST /v1/instagram/comments` |
| Compact MCP | `socq_execute` 与 `endpoint: "instagram/comments"` |
| Typed MCP | `socq_instagram_comments` |
| 精确工具 URL | `https://api.socq.ai/mcp?tools=instagram_comments` |
| CLI | `socq instagram comments` |

使用上面的 JSON 请求正文作为 `socq_execute` 的嵌套 `input` 对象，或将相同的字段直接传递给Typed 工具。Typed 工具还接受 `_wait_seconds`、`_result_limit` 和 `_idempotency_key`。

<Info>
  成功的采集将返回 `task_id`。继续 `socq_get_task` 直到任务成功，然后从 `results.items` 读取规范化记录。
</Info>

有关完整的提交和轮询流程，请参阅[MCP 快速入门](/zh/integrations/mcp)。
{/* mcp-usage:end */}

所有返回评论均按实际结果数计费。此端点不接受结果计数限制。
