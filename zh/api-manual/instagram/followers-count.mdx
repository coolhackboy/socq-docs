# Instagram 关注者计数

采集 Instagram 个人资料的公共关注者和关注计数。

## 端点

```http
POST https://api.socq.ai/v1/instagram/followers-count
```

## 请求正文

```json
{
  "usernames": ["instagram"]
}
```

{/* mcp-usage:start */}
## 与 MCP 一起使用

| 接口 | 值 |
| --- | --- |
| REST | `POST /v1/instagram/followers-count` |
| Compact MCP | `socq_execute` 与 `endpoint: "instagram/followers-count"` |
| Typed MCP | `socq_instagram_followers_count` |
| 精确工具 URL | `https://api.socq.ai/mcp?tools=instagram_followers_count` |
| CLI | `socq instagram followers-count` |

使用上面的 JSON 请求正文作为 `socq_execute` 的嵌套 `input` 对象，或将相同的字段直接传递给Typed 工具。Typed 工具还接受 `_wait_seconds`、`_result_limit` 和 `_idempotency_key`。

<Info>
  成功的采集将返回 `task_id`。继续 `socq_get_task` 直到任务成功，然后从 `results.items` 读取规范化记录。
</Info>

有关完整的提交和轮询流程，请参阅[MCP 快速入门](/zh/integrations/mcp)。
{/* mcp-usage:end */}

## 结果项

任务成功后，调用`GET /v1/tasks/{task_id}`读取`results`下分页的个人资料指标记录。

```json
{
  "id": "25025320",
  "platform": "instagram",
  "resource": "followers-count",
  "type": "profile_metrics",
  "username": "instagram",
  "name": "Instagram",
  "url": "https://www.instagram.com/instagram",
  "avatar_url": "https://example.com/avatar.jpg",
  "is_verified": true,
  "is_private": false,
  "metrics": {
    "followers_count": 685829413,
    "following_count": 251,
    "posts_count": null
  },
  "created_at": null,
  "collected_at": "2026-07-08T08:59:45",
  "extra": {
    "snapshot_time": "2026-07-08 - 02:02"
  }
}
```

您可以在同一字段中提交多个用户名：

```json
{
  "usernames": ["instagram", "creators"]
}
```
