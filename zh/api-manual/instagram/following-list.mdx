# Instagram 关注列表

```json
{"usernames": ["instagram"], "results_limit": 100}
```

## 与 MCP 一起使用

| 接口 | 值 |
| --- | --- |
| REST | `POST /v1/instagram/following-list` |
| Compact MCP | `socq_execute` 与 `endpoint: "instagram/following-list"` |
| Typed MCP | `socq_instagram_following` |
| CLI | `socq instagram following` |

只能返回对公共个人资料可见的关系。
