# TikTok 搜索

使用稳定的过滤器搜索公共视频。

```json
{
  "query": "python tutorial",
  "results_limit": 50,
  "sort_by": "relevance",
  "published_within": "month"
}
```

{/* mcp-usage:start */}
## 与 MCP 一起使用

| 接口 | 值 |
| --- | --- |
| REST | `POST /v1/tiktok/search` |
| Compact MCP | `socq_execute` 与 `endpoint: "tiktok/search"` |
| Typed MCP | `socq_tiktok_search` |
| 精确工具 URL | `https://api.socq.ai/mcp?tools=tiktok_search` |
| CLI | `socq tiktok search` |

使用上面的 JSON 请求正文作为 `socq_execute` 的嵌套 `input` 对象，或将相同的字段直接传递给Typed 工具。Typed 工具还接受 `_wait_seconds`、`_result_limit` 和 `_idempotency_key`。

<Info>
  成功的采集将返回 `task_id`。继续 `socq_get_task` 直到任务成功，然后从 `results.items` 读取规范化记录。
</Info>

有关完整的提交和轮询流程，请参阅[MCP 快速入门](/zh/integrations/mcp)。
{/* mcp-usage:end */}

|领域|接受的值|
| --- | --- |
| `sort_by` | `relevance`、`date`、`likes` |
| `published_within` | `day`、`week`、`month`、`three_months`、`six_months` |

`results_limit` 默认为 100。

结果使用[TikTok 视频](/zh/api-manual/tiktok/videos) 中记录的归一化结构，其中 `resource` 设置为 `search`。如果可用，提交的查询将在 `extra.search_query` 下返回。
