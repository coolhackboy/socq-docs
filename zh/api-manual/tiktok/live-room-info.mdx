# TikTok 直播间信息

```json
{"room_id": "7523685855395842871", "user_id": "6742945285876515845"}
```

## 与 MCP 一起使用

| 接口 | 值 |
| --- | --- |
| REST | `POST /v1/tiktok/live-room-info` |
| Compact MCP | `socq_execute` 与 `endpoint: "tiktok/live-room-info"` |
| Typed MCP | `socq_tiktok_live_room_info` |
| CLI | `socq tiktok live-room-info` |

结果包括实时状态、标题、封面图片、所有者、观看者计数和点赞计数。
