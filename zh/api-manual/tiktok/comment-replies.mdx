# TikTok 评论回复

采集公开 TikTok 评论的回复。结果会转换为统一结构，并按实际保存的记录数计费。

```json
{
  "url": "https://www.tiktok.com/@scout2015/video/6718335390845095173",
  "comment_id": "1234567890",
  "results_limit": 20
}
```

{/* mcp-usage:start */}
## 通过 MCP 使用

| 接口 | 值 |
| --- | --- |
| REST | `POST /v1/tiktok/comment-replies` |
| Compact MCP | `socq_execute`，并设置 `endpoint: "tiktok/comment-replies"` |
| 类型化 MCP | `socq_tiktok_comment_replies` |
| 精确工具 URL | `https://api.socq.ai/mcp?tools=tiktok_comment_replies` |
| CLI | `socq tiktok comment-replies` |

使用 `socq_execute` 时，将上面的 JSON 请求体作为 `input` 对象传入；也可以将相同字段直接传给类型化 MCP 工具。类型化 MCP 工具还支持 `_wait_seconds`、`_result_limit` 和 `_idempotency_key`。

<Info>
  提交成功后会返回 `task_id`。继续调用 `socq_get_task` 直到任务成功，然后从 `results.items` 读取标准化记录。
</Info>

完整的提交和轮询流程请参阅 [MCP 快速开始](/zh/integrations/mcp)。
{/* mcp-usage:end */}
