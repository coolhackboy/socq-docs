# TikTok Shop 用户橱窗

采集 TikTok 用户的公共商店展示柜中展示的产品。

## 请求示例

```json
{
  "username": "mrtiktokreviews",
  "results_limit": 100,
  "region": "US"
}
```

{/* mcp-usage:start */}
## 与 MCP 一起使用

| 接口 | 值 |
| --- | --- |
| REST | `POST /v1/tiktok-shop/user-showcase` |
| Compact MCP | `socq_execute` 与 `endpoint: "tiktok-shop/user-showcase"` |
| Typed MCP | `socq_tiktok_shop_user_showcase` |
| 精确工具 URL | `https://api.socq.ai/mcp?tools=tiktok_shop_user_showcase` |
| CLI | `socq tiktok-shop user-showcase` |

使用上面的 JSON 请求正文作为 `socq_execute` 的嵌套 `input` 对象，或将相同的字段直接传递给Typed 工具。Typed 工具还接受 `_wait_seconds`、`_result_limit` 和 `_idempotency_key`。

<Info>
  成功的采集将返回 `task_id`。继续 `socq_get_task` 直到任务成功，然后从 `results.items` 读取规范化记录。
</Info>

有关完整的提交和轮询流程，请参阅[MCP 快速入门](/zh/integrations/mcp)。
{/* mcp-usage:end */}

一项任务接受一个用户名。 `@` 前缀是可选的。
